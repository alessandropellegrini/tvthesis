\chapter{Introduction}
\label{ch:introduction}

Distributed computing has become an essential paradigm in modern software
engineering. As systems grow in complexity and scale, the need for efficient
distributed algorithms becomes increasingly critical~\cite{tanenbaum2007}.

The main contributions of this thesis are:
\begin{itemize}
    \item A new algorithm for consensus.
    \item A protocol for efficient data replication across distributed nodes.
    \item An extensive experimental evaluation of the proposed solutions.
\end{itemize}

The remainder of this thesis is organized as follows. Chapter~\ref{ch:background}
provides the necessary background. Chapter~\ref{ch:proposal} presents our
proposed approach. Chapter~\ref{ch:conclusions} draws conclusions and outlines
future work.



\chapter{Background}
\label{ch:background}

In this chapter, we review the fundamental concepts and related work that
form the basis of our research.

\section{Distributed Systems}

A distributed system is a collection of independent computers that appears to
its users as a single coherent system~\cite{tanenbaum2007}. The key challenges
in designing such systems include:
\begin{enumerate}
    \item Achieving consensus among distributed nodes.
    \item Ensuring fault tolerance and availability.
    \item Maintaining consistency across replicated data.
\end{enumerate}

\section{Consensus Algorithms}

Consensus algorithms are fundamental building blocks for distributed
systems. The most well-known algorithms include Paxos~\cite{lamport1998}
and Raft~\cite{ongaro2014}.

\section{System Model}

We consider a distributed system composed of $n$ nodes interconnected by
a network. Each node $i$ maintains a local state $s_i$ and communicates
with other nodes through message passing. The system satisfies the
following properties:
%
\begin{equation}
\label{eq:consistency}
\forall i, j \in \{1, \ldots, n\}: s_i = s_j \text{ at quiescence}
\end{equation}

\section{Algorithm Design}

Our algorithm proceeds in rounds. In each round $r$, every node $i$:
\begin{enumerate}
    \item Broadcasts its current state $s_i^{(r)}$ to all other nodes.
    \item Collects the states from a majority of nodes.
    \item Updates its state according to the deterministic merge function.
\end{enumerate}
